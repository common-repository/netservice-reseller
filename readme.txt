== Description ==

**پلاگین نت سرویس برای نمایندگان رسمی**

این پلاگین برای نمایندگان رسمی نت سرویس تهیه و توزیع شده
[برای اخذ نمایندگی اینجا کلیک کنید](https://netservice.shop/resellers)

## خدمات ارائه شده

* گواهینامه SSL
* وب اپلیکیشن
* بهینه سازی و سئو
* پیامک انبوه
* گوگل مارکتینگ
* ایمیل مارکتینگ
* تبلیغات پاپ آپ
* بک لینک

== Installation ==

نصب خودکار:
برای نصب خودکار بدون دانلود کردن از داشبورد وردپرس به قسمت افزونه ها رفته و روی افزودن کلیک کنید. در باکس جستجو عبارت "نت سرویس" را وارد کرده و افزونه را نصب و فعال کنید

نصب دستی:
افزونه را دانلود کرده و از صفحه افزونه ها در داشبورد وردپرس روی افزودن کلیک کنید و سپس بالای صفحه لود شده روی بارگذاری افزونه کلیک کرده و بعد از آپلود فایل زیپ آن را فعال سازی کنید

بعد از فعال سازی افزونه همه تنظیمات به صورت خودکار اعمال خواهد شد.

== Frequently Asked Questions ==

= برای اخذ نمایندگی باید چگونه اقدام کرد؟ =

برای اخذ نمایندگی باید به صفحه نمایدناگ در سایت نت سرویس رفته و با تکمیل فرم درخواست خود را مبنی بر اخذ نمایندگی ارسال کنید

= آیا سود من از فروش ثابت است؟ =

خیر ، میزان درآمد شما به میزان فروش و مقدار قیمتی که خودتان تعریف کرده اید دارد و نرخ از پیش تعیین شده ای برای نمایندگان قرار ندارد
